# Task Manager API

This is a **RESTful API** for managing tasks and users, built with **Express.js**.

## Features

- User management (create, read, update, delete)
- Task management (create, read, update, delete)
- JSON-based REST endpoints
- Error handling and input validation

## Installation

1. Clone the repo:

```bash
git clone https://github.com/Rafael123-droid/task_manager-api.git

2. Navigate to the project folder:
    cd task_manager_api

3. Install dependencies:    
    npm install

4. Start the server
    npm start
